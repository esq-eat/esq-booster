# ESQ Booster — Получить APK через GitHub (только телефон)

## Шаг 1 — Создать репозиторий на GitHub

1. Открыть **github.com** в браузере телефона
2. Войти в аккаунт (или зарегистрироваться — бесплатно)
3. Нажать **"+"** → **"New repository"**
4. Название: `esq-booster`
5. Visibility: **Private** (рекомендуется)
6. Нажать **"Create repository"**

---

## Шаг 2 — Загрузить файлы проекта

На странице нового репозитория:

1. Нажать **"uploading an existing file"**
2. Распаковать `ESQ-Booster.zip` на телефоне
3. Перетащить ВСЕ файлы и папки из распакованной папки
4. Нажать **"Commit changes"**

> ⚠️ Важно: загрузить нужно содержимое папки `ESQ-Booster/`,
> а не саму папку. GitHub должен видеть `app/`, `build.gradle`,
> `settings.gradle` в корне репозитория.

---

## Шаг 3 — Дождаться сборки APK

1. Перейти во вкладку **"Actions"** репозитория
2. Увидеть запущенный workflow **"Build ESQ Booster APK"**
3. Подождать ~5 минут (зелёная галочка = готово)

---

## Шаг 4 — Скачать APK

1. Нажать на завершённый workflow run
2. В разделе **"Artifacts"** внизу страницы
3. Нажать **"ESQ-Booster-debug"**
4. Скачается ZIP с APK внутри
5. Распаковать → установить `app-debug.apk`

---

## Установка APK на телефон

Настройки → Безопасность → **Установить из неизвестных источников** → включить для браузера/файлового менеджера

---

| Target Android | 14 (API 34) |
